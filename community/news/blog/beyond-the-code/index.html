Beyond the Code

There is an exciting conference coming up this fall. Beyond the Code will be happening on September 22, 2016 in Toronto this year! Beyond the Code is a positive, solutions-driven conference for anyone interested in diversifying the technology sector. It’s a great opportunity for all students interested in computer science and related fields.

Details, costs, and transportation are posted in this Google Doc. I will be updating it so don’t forget to check back often.

Please purchase your tickets and complete the form by September 10th.

If you have any questions please email me at elisa.kazan@ccss.carleton.ca.

Hope to see you all there!
